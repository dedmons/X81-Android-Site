Overview

This application is just a simple hello world app that adds 2 extra TextViews and a Button. The added views get their text from the strings resource XML file. When launched the app sets the buttons action to a function that will start a Timer that animates the TextViews to random parts of the screen and changes the buttons text. Then it sets the buttons action to a function that disables the timer, resets the TextViews, and resets the buttons text.
