Overview

This application is the start to a secure note taking and sharing platform. In this version the user creation, authentication, and note creation is implemented on top of an SQLite database. User passwords are secured using AES128 encryption and encoded in Base64 for storage. The master key is derived from the devices unique id.

The user registration screen puts the password through simple check and makes sure the confirmation is correct. The login screen gives error feedback based on your input and the data in the database.

Notes are stored based on the user who creates them.
